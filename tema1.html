Загальна структура документу HTML

Загальна структура документу - це теги, які необхідні для побудови веб-сторінки.
Для побудови веб сторінки необхідно внести наступні теги:

Title page

<!DOCTYPE html>
Оголошення типу документа
<html lang="language-Country">
Загальний тег для веб сторінки. Визначення мови сторінки і країни
<head>
Тег в розділі якого оголошується інформація про сторінку
<meta charset="charset_UTF-8">
Оголошення кодування сторінки, тобто які символи буде використовувати сторінка
<meta name="viewport" content="wdth=device-width, initial-scale=1.0">
Область перегляду, оголошується що область перегляду буде мати ширину вікна пристрою, з якого переглядають сторінку, встановлюється початковий маштаб сторінки розміром 1.0(ширини області перегляду пристрою)
<meta name="author" content="author_name">
Оголошується автор сторінки і вказується авторство
<meta name="description" content="descrition_content">
Оголошується опис сторінки і вказується загальний опис сторінки
<meta name="keywords" content="keyword1, keyword2, keyword3">
Оголошуються ключові слова сторінки і вказуються ключові слова стосовно вмісту сторінки
<link rel="stylesheet" href="name_of_css_file.css">
Підключається сторінки з каскадними таблицями стилів для поточної веб-сторінки
<link rel="icon" type="image/x-iocon" href="path_to_icon">
Встановлюється іконка, яка буде відображатися у вкладці браузера
<title>Title content</title>
Встановлюється заголовок сторінки, який буде відображатися на вкладці браузера
<style>
Оголошується тег стилів для поточної сторінки
.class_name{
Оголошується клас стилів
class_style_atribute1;
Перший атрибут стилю класу
class_style_atribute2;
Другий атрибут стилю класу
...
Можливі атрибути стилю класу
class_style_atributeN;
N-й атрибут стилю класу
}
Завершення оголошеного класу стилів
#id_name{
Оголошення стилю елемента з айді
id_style_atribute1;
Перший атрибут стилю елемента айді
id_style_atribute2;
Другий атрибут стилю елемента айді
...
Можливі атрибути стилю елемента айді
id_style_atributeN;
N-й атрибут стилю елемента айді
}
Завершення оголошеного елементу айді з стилями
element_name{
Оголошення стилів конкретного елемента сторінки
element_style_atribute1;
Перший атрибут стилю елемента сторінки
element_style_atribute2;
Другий атрибут стилю елемента сторінки
...
Можливі атрибути стилю елемента сторінки
element_style_atributeN;
N-й атрибут стилю елемента сторінки
}
Завершення оголошених стилів елементу сторінки
@media display only (min-width:500px){
Оголошення контролю ширини вікна відображення сторінки, якщо мінімільна ширина 500 пікселів то виконати те, що оголошено в вкладених елементах
element_name1, ... ,element_nameN{
Для вказаних елементів виконати оголошене в вкалденому елементі
width:100%;
При ширині 500 пікселів встановити ширину вказаних елементів на 100%
}Завершення оголошеного для елементів сторінки атрибуту стилів
}
Завершення налаштування для адаптивності веб-сторінки
</style>
Завершення тегу стилів для поточної сторінки
</head>
Завершення тегу хед і завершення оголошення інфморації про сторінку
<body>
Відкриття тегу тіла сторінки, в який вклидається видиме наповнення веб-сторінки
<header>
Відкриття тегу шапки сторінки
<h1>Header_content<h1>
Вказування заголовку описуючого всю веб-сторінки, розміщеного в шапці сторінки
</header>
Закриття тегу шапки сторінки
<nav>
Оголошення горизонтальної панелі навігації для сторінок веб-сайту
<span><a href="path_to_page1.html" target="iframe_name">Home</a></span>
Перший пункт навігації, посилання на домашню сторінку, яка відкриється в фреймі
<span><a href="path_to_page2.html" target="iframe_name">About</a></span>
Другий пункт навігації, посилання на сторінку "Про нас", яка відкриється в фреймі
<span><a href="path_to_page3.html" target="iframe_name">Contacts</a></span>
Третій пункт навігації, посилання на сторінку "Контакти", яка відкриється в фреймі
</nav>
Закриття тегу горизонтальної панелі навігації сторінок веб-сайту
<nav>
Оголошення вертикальної панелі навігації для сторінок веб-сайту
<div><a href="path_to_page4.html" target="iframe_name">Page1</a></div>
Блок з посиланням на сторінку 1, яка відкриється в фреймі
<div><a href="path_to_page5.html" target="iframe_name">Page2</a></div>
Блок з посиланням на сторінку 2, яка відкриється в фреймі
...
Можливі блоки з посиланнями на сторінки, яка відкриється в фреймі
<div><a href="path_to_pageN.html" target="iframe_name">PageN</a></div>
Блок з посиланням на сторінку N, яка відкриється в фреймі
</nav>
Закриття тегу вертикальної панелі навігації сторінок сайту
<section>
Оголошення тематичного блоку сторінки
<iframe src="iframe_page.html" style="width:100%;height:2500px;border:none;" name="iframe_name"> </iframe>
Оголошення фрейму, який відображає сторінки, на які посилаються навігаційні панелі
</section>
Закриття тематичного блоку сторінки
<footer>
Відкриття блоки підвалу сторінки
<nav>
Оглошення панелі навігаії карти сайту в підвалі сторінки
<ul>
Оголошення списку сторінкок присутніх на сайті
<li><a href="path_to_page1.html">Home</li>
Оголошення пункту списку карти сайту, з посиланням на домашню сторінку
<li><a href="path_to_page2.html">About</li>
Оголошення пункту списку карти сайту, з посиланням на сторінку "Про нас"
<li><a href="path_to_page3.html">Contacts</li>
Оголошення пункту списку карти сайту, з посиланням на сторінку "Контакти"
</ul>
Закриття тегу списку сторінок
</nav>
Закриття панелі навігації карти сайту в підвалі сторінки
<nav>
Оголошення другої панелі навігації карти сайту в підвалі сторінки
<ul>
Оголошення списку сторінок присутніх на сайті
<li><a href="path_to_page4.html">Page1</li>
Оголошення пункту списку карти сайту, з посиланням на сторінку 1
<li><a href="path_to_page5.html">Page2</li>
Оголошення пункту списку карти сайту, з посиланням на сторінку 2
...
Можливі пункти списку посилань карти сайту, на сторінки
<li>a href="path_to_pageN.html">Page2<PageN</li>
Оголошення пункту cписку карти сайту, з посиланням на сторінку N
</ul>
Закриття тегу списку сторінок для карти сайту
</nav>
Закриття тегу навігаційної панелі для карти сайту
<p>© Made by Company_name</p>Вкладення абзацу з правами на інформацію сторінки, які належать визначеній компанії
</footer>
Закритяя тегу підвалу сторінки
</body>
